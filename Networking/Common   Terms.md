# SOC Analyst Quick Reference

| Name     | Explanation                                                                                                                                                             |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Router   | Forwards packets between networks, finds best path.<br>can connect different network.                                                                                   |
| Switch   | Connects devices, forwards data using MAC addresses.<br>can only connect internal network systems                                                                       |
| Hub      | Broadcasts data to all devices, no filtering.                                                                                                                           |
| LAN      | Local network in small area like office/home.                                                                                                                           |
| WAN      | Network across large geographic areas.                                                                                                                                  |
| PAN      | Small personal network like Bluetooth devices.                                                                                                                          |
| ISP      | Internet Service provider.                                                                                                                                              |
| IDS      | Detects suspicious network/host activity.<br>Intrusion detection system.                                                                                                |
| IPS      | Detects and blocks malicious activity.<br>Intrusion Protection System                                                                                                   |
| NAT      | Translates private IPs to public IPs for internet.<br>Network Address Translator                                                                                        |
| Firewall | Filters traffic based on security rules.                                                                                                                                |
| SIEM     | Centralizes logs and generates alerts.<br>Security Information and Event Management                                                                                     |
| Honeypot | Decoy system to lure and detect attacks.                                                                                                                                |
| VPN      | Secures remote connections via encryption.<br>Virtual private network                                                                                                   |
| NAC      | Controls network access based on policies.<br>Network access control system                                                                                             |
| DLP      | Prevents leakage of sensitive data.<br>Data loss prevention service                                                                                                     |
| LWDA     | Light weight directory access<br><br>**==Protocol jo organization ke users, groups aur devices ko manage aur authenticate karne ke liye use hota hai==**                |
| ISN      | Initial Sequence Number<br><br>A number that  a system sends as a syn request to server and the server sends a syn-ack request in return as client-ISN+1  as a response |
